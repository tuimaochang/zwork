可设置属性：

  • id - 编号
  • title - 标题
  • container - 所在容器，可以收用jQuery查询到的对象，可以接受一个标号id。
  • containerStyle - 所在容器样式
  • src - 页面地址
  • content - 静态页面内容
  • top - 相对于容器顶部的距离
  • left - 相对于容器左部的距离
  • width - 窗口宽度
  • height - 窗口高度
  • minHeight - 最小高度
  • minWidth - 最小宽度
  • maxWidth - 最大宽度
  • maxHeight - 最大高度
  • maxable - 是否允许最大化
  • resizable - 是否允许充值大小
  • scroll - 是否出现滚动条
  • iframe - 是否启用iframe
